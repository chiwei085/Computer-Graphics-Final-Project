# Computer Graphics Final Project 2026

Theme: AI Future World

Design a 3D scene that represents an AI-powered future world. Your scene should 
include various animated objects/shapes, textures, lighting effects, and at least one AI
related character or object.

## Grading

- Scene Creativity (20%)
- Technique Difficulty (20%)
- Required Features (2 - 6 terms, 50%)
- Code, Report, and Demo (10%)

## Project Requirements

### Texture Mapping (10%)

Use 4 or more textures

- Textures cannot be the same as course examples.  
- Apply textures to different types of objects.  
- At least one texture must be used on an AI-related object.  

### AI theme-related Object Model (10%)

Load at least one external OBJ model related to AI or future technology.

- Cannot use provided example models.  
- Must include animation.  

### Animation (20%)

- Self Rotation (5%): Object_A rotates on its own axis.
- Orbit Motion (5%): Object_B revolves around Object_A with a cosine/sine wave movement.
- Custom Animation (10%): Create at least two additional animation. 

### Light and Shadow (10%)

Implement lighting and shadow effects.

- At least 2 light sources.
- Shadow must be visible.

## Note

- Do not use external game engines.
- Only use OpenGL (legacy FreeGLUT) and [stb_image](https://raw.githubusercontent.com/nothings/stb/refs/heads/master/stb_image.h)

## Personal Code style

- C++20, no any C style
- Google style comments
- Use clang-format
