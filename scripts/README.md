# scripts — Asset Pipeline

Pre-processing scripts that convert public-domain 3D models into the OBJ files
used by the runtime. These scripts run **offline** (not at runtime) and require
Python with [uv](https://docs.astral.sh/uv/).

## Setup

```bash
cd scripts
uv sync          # creates .venv/ and installs all deps
```

## Usage

### 1. Download raw assets

```bash
uv run python fetch_assets.py fetch          # download all auto entries
uv run python fetch_assets.py fetch robonaut2 ingenuity   # specific keys only
uv run python fetch_assets.py list           # print the curated shortlist
```

Downloads GLB files into `assets/raw/` and writes `assets/CREDITS.md` with
sha256 checksums and license info.

### 2. Convert GLB → OBJ

```bash
uv run python convert_glb_to_obj.py                  # convert all of assets/raw/
uv run python convert_glb_to_obj.py --scale 1.5      # scale to different bound
uv run python convert_glb_to_obj.py path/to/file.glb # single file
```

Outputs normalized OBJ + MTL into `assets/obj/`. Handles
`KHR_draco_mesh_compression` (the NASA models use Draco).

> **Note:** `assets/obj/Robonaut_2.obj` (~18 MB) is excluded from the
> repository. Run step 2 to regenerate it after cloning.
